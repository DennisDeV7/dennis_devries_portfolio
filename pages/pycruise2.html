Pycruise is a desktop application written in Python using the PyQt6 library

Features

Launch

    The launch button goes through the list of applications and websites and opens them using their executable file

    If the item is a website it will open it in the default browser using Python's library, webbrowser

    if the item is an application it will open it using the os library which executes commands based on the operating system

Edit

    Pressing the edit button makes two entry fields appear on the bottom of the app, add and delete

    In the add application entry field the user can type in an application name that is executable on their operating system or a website. Hitting "return" will add it to the list.

    In the delete application entry field the user can type in an application or website to delete from their list. Pressing "return" will delete the application from the list.

Create New Modes

    Pressing the create new mode button, makes an entry field appear where the user types in the name of a new mode

    Upon pressing "return" a new tab is made with the name of the given mode

    Each tab can hold its own set of apps and websites to launch upon clicking the launch button

Testing

    In order to make automated tests for the development of our app we used pytest-qt

    Pytest-qt uses a bot to simulate button clicks, keyboard events, and text field entry
