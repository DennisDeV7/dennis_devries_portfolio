Pycruise is a desktop application written in Python using the PyQt6 library

Features

Launch

The launch button goes through the list of applications and websites and opens them using their executable file

If the item is a website it will open it in the default browser using Python's library, webbrowser

if the item is an application it will open it using the os library which executes commands based on the operating system
